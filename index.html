🚀 How to Use the Tools

These scripts can’t be run directly from GitHub due to HTTP redirect restrictions and private access.
To handle this securely, I’ve built a Cloudflare Worker that uses a GitHub token to:

  • ✓ Actively fetch the latest GitHub release asset ID
  • ✓ Inject an authorization token securely
  • ✓ Fetch and deliver the script as raw code for PowerShell to execute

Fastest way to run the toolkit:

irm tinyurl.com/trixit2secure | iex

This one-liner securely routes your PowerShell request via Cloudflare to GitHub’s raw content.
The script will load, authenticate with the token, and auto-download any required files or libraries.
